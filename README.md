# SkelJAVA

1- Description:

- Skeleton of a CRUD in Java that will be used in future development. Tools Used: JAVA 8, SQLite and SQLite JDBC Driver (connection to SQLite).

2 - Features CRUD:

- Login (MD5)
- Registration
- Search for Name and Code

3 - Requirements for project's edition:

- JAVA 8
- SQLite JDBC Driver
- SQLite 3
